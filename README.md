# Compilation Output for X-Plane Remote Control Protocol Specification

This branch contains compiled documentation files of [XPRC](https://codeberg.org/dneuge/xprc-protocol) which,
in addition to holding copies of released versions, allows easy access to unreleased (untagged) revisions.

> [!warning]
> This branch is being automatically overwritten by periodically executed CI jobs. Due to the history being
> lost/rewritten it is recommended not to synchronize with this branch but only download it (or specific files
> of it) as needed.

Please refer to the `main` branch and respective files/directories for license information.
